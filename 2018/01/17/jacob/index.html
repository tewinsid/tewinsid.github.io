java操纵office

原理

####组件对象模型（component Object Model）：微软的一套软件组件的二进制接口的标准，使动态创建成为可能。实际上com是一种与语言无关的对象实现方式，可以再不同环境下被复用，com允许复用对象而不必了解对象内部实现细节，因此组件实现者必须提供良好的接口定义以屏蔽实现细节，通过引用计数组件对象自己负责构造和析构，从而屏蔽里不同语言之间的内存差异。com已经部分被.net取代。

####动态链接库（dll）
是微软在windows系统中实现共享函数库概念的一种实现方式，扩展名.dll、.OCX(包含activeX控制的库）或者.DRV(旧式系统驱动程序)

####动态链接
二进制文档，与exe文档相同。把一些经常公用的代码制作成dll文档，当可执行文件调用到dll文档内的函数时，windows才会把dll加载到内存中，内存浪费的情形可大幅降低。静态连接库直接连接到可执行文件（exe）。将dll加载到内存后，会得到一组函数的地址，通过call调用想用的函数

####jacob介绍
目的：通过java调用COM组件实现操纵Office
COM组件提供了一种与其他语言交互的方式，叫做自动化（automation），使用自动化，调用COM中暴露的方法，主要有两种机制，早起绑定和晚期绑定
早起绑定：显示的声明对象、数据类型、编译器获取足够的信息进行链接和优化，这样通常可以获得更好的性能，IBM的RJCB项目，性能高，需要了解COM组件更多的实现细节，工作量大
晚期绑定：通过IDispatch接口实现，类似于java反射，可以按照名称和id进行方法调用，这种设计主要是支持脚本语言操纵COM，性能较差，灵活，简单，不必关注动态链接库的细节

操纵word常用操作

  • 表格操作
目标对象 操作 name params return descript
doc get “Tables” null tables 全部表格对象
tables get “Count” null 表格数量 获取表格数量
tables call “Item” new Variant(tableIndex) table 获取第n个表格
table get “Columns” null cols 列对象
cols get “Count” null 表格列数量 获取特定表格列个数
table get “Rows” null rows 行对象
rows get “Count” null 表格行数量 获取特定表格行个数
rows get “Item” new Variant(rowIndex) row 某一行对象
rows call “Add” new Variant(row) null 在指定行前添加一行
table call “Cell” new Variant(cellRowIdx),new Variant(cellColIdx) cell 指定单元格
cell call “Select” null null 选中指定单元格
selection get “Font” null font 得到字体对象
font put “Name” new Variant(“宋体”) null 设置字体
font put “Size” new Variant(12) null 设置字号
font put “Color” color null 设置字体颜色
font put “Bold” new Variant(bold) null 设置加粗
font put “Underline” new Variant(underLine) null 设置下划线
selection put “Text” 内容 null 填充内容
  • 加密操作
目标对象 操作 name params return descript
doc get “ProtectionType” null String protectionType 是否加密-1未加密
doc call “Protect” new Variant(2), new Variant(true), pwd null 加密
doc call “Unprotect” pwd null 解密
  • 查找操作
目标对象 操作 name params return descript
word get “Selection” null selection 当前光标
selection call “Find” null find 得到从当前光标开始的查找对象
find put “Text” toFindText null 设置查找内容
find put “Forward” “True” null 设置向前查找
find put “MatchCase” “True” null 设置大小写匹配
find call “Execute” null Variant 查找并选中
selection get “Range” null range 区间对象
range call “SetRange” start + “”, end + “” null 选中start到end
range get “Text” null Variant 选中区间文本
  • 光标操作
目标对象 操作 name params return descript
word get “Selection” null selection 当前光标
selection put “Text” newText null 替换文本
selection call “MoveRight” null null 光标右移
selection call “MoveLeft” null null 光标左移
selection call “MoveUp” null null 光标上移
selection call “MoveDown” null null 光标下移

jacobFAQ

  • 找不到jacob.dll，配置路径问题（确认jacob.dll配置正确）

  • 找不到资源文件，资源文件路径问题（查看程序中资源文件路径的设置，并检查资源文件的存放路径）

  • jacob.dll或jacob.jar对应版本错误

jacob的操作大部分可以通过vba演化过来，可通过录制宏的方式得到想要的操作。
